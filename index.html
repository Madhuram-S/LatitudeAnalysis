Weather Analysis By Latitude
Lattitude vs. X

Can we prove with data that it gets hotter as we come closer to Equator?

The purpose of this project is to prove it does...

We will use Python requests, APIs, and JSON traversals to answer this fundamental question and explore the effect of Latitude on Maximum Temperature, Humidity, Cloudiness, and Wind Speed.

To accomplish this, we use Openweather API to obtain Temperature, Humidity, Cloudiness and Wind Speed for over 500 unique cities across the world.

After assembling the data, we explore and analyse the below relationships using scatter plots.

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude

The results of the analysis are published in this site along with source data and generated Visualizations
